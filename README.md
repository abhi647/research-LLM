# The Big Bang Hoax - Research Community

An interactive platform exploring alternative cosmological theories, featuring a community discussion feed and an AI-powered research assistant.

## Features

-   **Community Hub**: A feed of the latest alternative physics theories, including the featured article "THE BIG BANG HOAX: A Lineal Fallacy" by Dim O. Njaka.
-   **AI Research Assistant**: A chat interface to explore documents and discuss theories (Mock RAG implementation).
-   **Resource Library**: Access to research papers and datasets (Mocked).

## Run Locally

**Prerequisites:** Node.js (v18+)

1.  **Clone the repository**
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Deploy on Vercel

The easiest way to deploy this app is to use the [Vercel Platform](https://vercel.com/new).

1.  Push this code to a GitHub repository.
2.  Go to Vercel and sign up/login.
3.  Click **"Add New..."** -> **"Project"**.
4.  Import your GitHub repository.
5.  Vercel will detect `Vite` automatically.
6.  Click **Deploy**.
